# Задача 2: Бесконечный цикл и неправильная сумма

## Описание
Программа должна вычислить сумму чисел от 1 до 5, но результат оказывается неправильным (либо программа зависает, либо выдаёт неверное значение). Ваша задача — с помощью отладчика найти причину ошибки и исправить код.

## Условие
```java
public class DebugTask2 {
    public static void main(String[] args) {
        int sum = calculateSum(5);
        System.out.println("Sum: " + sum);
    }
    public static int calculateSum(int n) {
        int sum = 0;
        for (int i = 1; i <= n; i--) {
            sum += i;
        }
        return sum;
    }
}
```

## Ожидаемый результат
Программа должна вывести сумму чисел от 1 до 5:
```
Sum: 15
```

## Фактическое поведение
При запуске программа либо:
- Зависает (бесконечный цикл) и не завершается, либо
- Завершается, но выводит неправильную сумму.

## Задание
1. Запустите программу в режиме отладки (Debug).
2. Установите точку останова (breakpoint) внутри цикла.
3. Проанализируйте значения переменных на каждом шаге выполнения.
4. Найдите причину некорректной работы.
5. Исправьте код так, чтобы он работал согласно ожиданию.
6. Запишите процесс отладки и исправления на видео.

## Подсказки для отладки
- Обратите внимание на то, как изменяется счётчик цикла `i` на каждой итерации.
- Проверьте условие завершения цикла.
- Посмотрите, достигает ли цикл когда-нибудь условия выхода.
