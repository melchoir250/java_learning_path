# Задача 8: Сравнение чисел с плавающей точкой

## Описание
Программа должна сравнить два числа: результат умножения `0.1 * 3` и число `0.3`. Математически эти значения равны, но программа выводит "Not Equal". Ваша задача — с помощью отладчика найти причину такого поведения и исправить код.

## Условие
```java
public class DebugTask8 {
    public static void main(String[] args) {
        double a = 0.1 * 3;
        double b = 0.3;
        if (a == b) {
            System.out.println("Equal");
        } else {
            System.out.println("Not Equal");
        }
    }
}
```

## Ожидаемый результат
Программа должна вывести:
```
Equal
```

## Фактическое поведение
При запуске программа выводит:
```
Not Equal
```

## Задание
1. Запустите программу в режиме отладки (Debug).
2. Установите точку останова (breakpoint) на строке с условием `if`.
3. Проанализируйте значения переменных `a` и `b`.
4. Обратите внимание на их точные значения в отладчике.
5. Найдите причину, по которой `a` и `b` не равны.
6. Исправьте код так, чтобы он корректно сравнивал числа с плавающей точкой.
7. Запишите процесс отладки и исправления на видео.

## Подсказки для отладки
- Посмотрите на значения `a` и `b` в панели переменных отладчика.
- Обратите внимание на то, как числа с плавающей точкой представляются в памяти.
- Подумайте, какой способ сравнения `double` чисел является корректным.
