# Задача 9: Сравнение строк

## Описание
Программа должна проверить, равны ли две строки с одинаковым содержанием "hello". Однако при запуске программа выводит "Not Equal", хотя строки визуально одинаковы. Ваша задача — с помощью отладчика найти причину такого поведения и исправить код.

## Условие
```java
public class DebugTask9 {
    public static void main(String[] args) {
        String str1 = new String("hello");
        String str2 = new String("hello");
        if (str1 == str2) {
            System.out.println("Equal");
        } else {
            System.out.println("Not Equal");
        }
    }
}
```

## Ожидаемый результат
Программа должна вывести:
```
Equal
```

## Фактическое поведение
При запуске программа выводит:
```
Not Equal
```

## Задание
1. Запустите программу в режиме отладки (Debug).
2. Установите точку останова (breakpoint) на строке с условием `if`.
3. Проанализируйте значения переменных `str1` и `str2`.
4. Обратите внимание не только на содержимое строк, но и на их идентификаторы (ссылки) в отладчике.
5. Найдите причину, по которой условие не срабатывает.
6. Исправьте код так, чтобы он корректно сравнивал строки.
7. Запишите процесс отладки и исправления на видео.

## Подсказки для отладки
- Посмотрите на значения переменных `str1` и `str2` в панели переменных отладчика.
- Обратите внимание на их идентификаторы объектов (object ids) или адреса в памяти.
- Подумайте, чем отличается сравнение ссылок от сравнения содержимого объектов.
