# Задача 1: Выход за границы массива

## Описание
Программа должна напечатать все элементы массива, но при запуске выбрасывает исключение. Ваша задача — с помощью отладчика найти причину ошибки и исправить код.

## Условие
```java
public class DebugTask1 {
    public static void main(String[] args) {
        int[] numbers = {10, 20, 30, 40, 50};
        for (int i = 0; i <= numbers.length; i++) {
            System.out.println(numbers[i]);
        }
    }
}
```

## Ожидаемый результат
Программа должна вывести все пять элементов массива:
```
10
20
30
40
50
```

## Фактическое поведение
При запуске программа выбрасывает исключение и аварийно завершается.

## Задание
1. Запустите программу в режиме отладки (Debug).
2. Установите точку останова (breakpoint) в начале цикла.
3. Проанализируйте значения переменных на каждом шаге выполнения.
4. Найдите причину возникновения исключения.
5. Исправьте код так, чтобы он работал согласно ожиданию.
6. Запишите процесс отладки и исправления на видео.

## Подсказки для отладки
- Обратите внимание на значение счётчика цикла `i` на последней итерации.
- Сравните максимальный индекс массива с его длиной.
- Проверьте условие завершения цикла.
