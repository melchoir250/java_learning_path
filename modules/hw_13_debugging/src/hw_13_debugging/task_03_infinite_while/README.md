# Задача 3: Бесконечный цикл (зависание)

## Описание
Программа должна напечатать числа от 1 до 5, но при запуске зависает и не завершается. Ваша задача — с помощью отладчика найти причину ошибки и исправить код.

## Условие
```java
public class DebugTask3 {
    public static void main(String[] args) {
        int i = 1;
        while (i <= 5) {
            System.out.println("Number: " + i);
        }
    }
}
```

## Ожидаемый результат
Программа должна вывести числа от 1 до 5:
```
Number: 1
Number: 2
Number: 3
Number: 4
Number: 5
```

## Фактическое поведение
При запуске программа продолжает выполняться бесконечно и не завершается. В консоли может бесконечно печататься одно и то же число.

## Задание
1. Запустите программу в режиме отладки (Debug).
2. Установите точку останова (breakpoint) внутри цикла.
3. Проанализируйте значения переменных на каждом шаге выполнения.
4. Найдите причину бесконечного цикла.
5. Исправьте код так, чтобы он работал согласно ожиданию.
6. Запишите процесс отладки и исправления на видео.

## Подсказки для отладки
- Обратите внимание на то, меняется ли значение переменной `i` внутри цикла.
- Проверьте условие выхода из цикла.
- Понаблюдайте за выполнением нескольких итераций.
