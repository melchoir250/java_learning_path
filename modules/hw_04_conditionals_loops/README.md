# Домашнее задание: Условные операторы и циклы в Java

## Описание проекта

Этот проект посвящён практическому освоению управляющих конструкций Java: условных операторов `if-else` и `switch`, а также циклов `for`, `while`, `do-while` и операторов управления потоком `break` и `continue`.
Задачи направлены на формирование уверенных навыков работы с логикой принятия решений и повторяющимися действиями в программах.

## Основные темы и задачи

### Условные операторы (if-else)

Работа с логическими проверками и разветвлениями программы:
- Определение знака числа
- Поиск наибольшего из двух чисел
- Преобразование оценки 1–5 в текстовый формат
- Проверка числа на чётность
- Определение скидки по возрасту
- Оценка результата теста (по диапазонам баллов)

### Оператор switch

Применение конструкции `switch-case` для обработки фиксированных вариантов:
- Определение дня недели по номеру
- Расчёт стоимости билета по дню
- Перевод числовых оценок в буквенные (A–F)
- Обработка команд (start/stop/restart/status)
- Простой калькулятор на основе оператора

### Цикл for

Используется для повторения действий с известным количеством итераций:
- Вывод чисел, делящихся на 3
- Сумма чисел от 1 до n
- Таблица умножения
- Проверка числа на простоту
- Вывод чисел от 1 до 10

### Цикл while

Подходит для выполнения, пока условие истинно:
- Вычисление факториала
- Вывод всех чётных чисел до введённого значения
- Обратный отсчёт до 1

### Цикл do-while

Гарантирует хотя бы одно выполнение блока:
- Запрос положительного числа
- Проверка пароля
- Вывод чисел 1–10
- Завершение программы по команде “exit”
- Подсчёт количества цифр в числе

### Операторы break и continue

Контроль выполнения циклов:
- Прерывание ввода при отрицательном числе
- Пропуск чисел, делящихся на 3
- Вывод только положительных значений
- Завершение ввода строк по команде `stop`

## Ключевые концепции

### Логические конструкции
- Вложенные проверки
- Обработка вариантов с помощью `switch`
- Оптимизация условий

### Циклы
- Итерации с контролем счётчика
- Проверки внутри цикла
- Различия между `for`, `while`, `do-while`

### Управление потоком
- Использование `break` для немедленного выхода
- Применение `continue` для пропуска итерации

## Технические особенности
- Чтение пользовательского ввода через `Scanner`
- Работа со строками и числами
- Использование арифметических операций и операторов сравнения
- Написание простых алгоритмов для обработки данных
