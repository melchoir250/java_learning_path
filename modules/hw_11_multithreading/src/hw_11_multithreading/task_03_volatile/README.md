# Задача 3: Использование volatile

**Уровень сложности:** Средний

## Условие задачи

Создайте поток, который бесконечно увеличивает счетчик. В основном потоке через 2 секунды установите флаг stop = true,
чтобы остановить поток.

## Требования

- Создать поток, который в цикле увеличивает переменную-счетчик
- Поток должен работать, пока флаг `stop` не станет true
- Флаг `stop` должен быть объявлен с ключевым словом `volatile`
- В основном потоке через 2 секунды установить `stop = true`
- Вывести финальное значение счетчика

## Пример вывода

```
Счетчик: 123456789
Поток остановлен
```

## Что практикуется

- Понимание проблем видимости переменных между потоками (visibility)
- Использование ключевого слова `volatile` для гарантии видимости
- Корректная остановка потока через флаг (не используя `stop()`)
- Кеширование переменных в многопоточной среде

## Подсказки

1. **volatile**: гарантирует, что изменение переменной в одном потоке будет видно в других
2. **Без volatile**: поток может "не увидеть" изменение флага из-за кеширования
3. **Цикл**: `while (!stop) { counter++; }`
4. **Задержка**: `Thread.sleep(2000)` в основном потоке перед установкой флага
5. **Эксперимент**: попробуйте убрать `volatile` и объяснить результат
