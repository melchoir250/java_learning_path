# Задача 2: Создание двух потоков

**Уровень сложности:** Начальный

## Условие задачи

Создайте два потока. Один поток должен печатать "A", второй — "B", каждый по 5 раз с небольшой задержкой.

## Требования

- Создать два потока
- Первый поток выводит букву "A" 5 раз
- Второй поток выводит букву "B" 5 раз
- Между выводами каждого потока должна быть пауза (рекомендуется 500 мс)
- Запустить оба потока из метода `main`

## Пример вывода (возможные варианты)

```
A
B
A
B
A
B
A
B
A
B
```

или

```
A
A
B
A
B
B
A
B
A
B
```

## Что практикуется

- Работа с несколькими потоками одновременно
- Конкуренция потоков за вывод в консоль
- Планировщик потоков и недетерминированность выполнения
- Использование задержек для наблюдения за переключением контекста

## Подсказки

1. **Два потока**: можно создать два отдельных класса или один класс с параметром
2. **Задержка**: `Thread.sleep(500)` — полсекунды между выводами
3. **Неопределенность**: порядок вывода может меняться при каждом запуске
4. **Наблюдение**: попробуйте убрать задержку и посмотреть, что изменится
