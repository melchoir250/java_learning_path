# Задача 1: Создание одного потока

**Уровень сложности:** Начальный

## Условие задачи

Напишите программу, в которой создается отдельный поток, выводящий сообщение "Привет из потока!" 5 раз с паузой в 1
секунду между сообщениями.

## Требования

- Создать отдельный поток (наследование от `Thread` или реализация `Runnable`)
- Поток должен вывести "Привет из потока!" 5 раз
- Между выводами должна быть пауза в 1 секунду (1000 миллисекунд)
- Запустить поток из метода `main`

## Пример вывода

```
Привет из потока!
Привет из потока!
Привет из потока!
Привет из потока!
Привет из потока!
```

## Что практикуется

- Создание потока через класс `Thread` или интерфейс `Runnable`
- Запуск потока методом `start()`
- Использование `Thread.sleep()` для создания пауз
- Различие между основным потоком и дочерним потоком

## Подсказки

1. **Создание потока**:
    - Вариант А: создать класс, наследующий `Thread`, и переопределить `run()`
    - Вариант Б: создать класс, реализующий `Runnable`, и передать его в конструктор `Thread`

2. **Запуск потока**: используйте метод `start()`, а не `run()`

3. **Пауза**: `Thread.sleep(1000)` может выбросить `InterruptedException` — нужно обработать

4. **Наблюдение**: обратите внимание, что сообщения из потока могут перемежаться с выводом из `main`
