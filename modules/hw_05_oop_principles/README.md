# Домашнее задание: ООП принципы в Java

## Описание проекта

Этот модуль посвящён практическому применению ключевых принципов объектно-ориентированного программирования: инкапсуляции, наследования, полиморфизма и абстракции.
В рамках заданий вы проектируете структуры классов, выделяете общие сущности, создаёте интерфейсы и абстрактные классы, а также реализуете разное поведение объектов через единый интерфейс.

## Темы и содержание

### OOP Principles Practice
- Абстракция общих характеристик
- Инкапсуляция состояния и поведения
- Наследование и расширение функциональности
- Полиморфизм — единый интерфейс, разные реализации
- Создание иерархий классов
- Использование интерфейсов и абстрактных классов

### Практические задачи
- **Зоопарк** — разные животные с уникальным поведением
- **Домашние питомцы** — взаимодействие и уход
- **Ресторан** — горячие блюда и напитки с разными параметрами
- **Аквариум** — морские существа с разными способами передвижения
- **Ферма** — животные с потребностями и функциями
- **Ботанический сад** — растения с разными условиями ухода
- **Парк развлечений** — аттракционы и обслуживание
- **Музей** — экспонаты с историей и условиями хранения
