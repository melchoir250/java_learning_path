# Нахождение среднего значения

## 📌 Условие
Написать unit-тесты (JUnit 5) для метода из этой задачи.

## ✅ Что проверить
- Позитивные сценарии (валидные входные данные)
- Граничные значения (edge cases)
- Негативные сценарии (ошибочные входные данные), если применимо
- Исключения (если метод должен их бросать)

## 📂 Файлы
- Production-код: `src/main/java/...`
- Тесты: `src/test/java/...`

## 🚀 Запуск
```bash
mvn test
```
