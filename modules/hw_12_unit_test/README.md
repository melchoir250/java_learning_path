
# 🧪 Модуль 12: Юнит-тестирование (JUnit)

## 📋 Описание

Модуль содержит 20 практических задач для освоения юнит-тестирования с использованием **JUnit**. Каждая задача включает реализацию метода и написание тестов, покрывающих различные сценарии: позитивные кейсы, граничные значения и обработку исключений.

---

## 🏛️ Структура модуля

```
hw_12_unit_test/
├── docs/tasks/              # Документация с описанием задач
│   └── task_*_*/README.md   # README для каждой задачи
├── src/main/java/           # Исходный код (реализация методов)
│   └── hw_12_unit_test/
│       └── task_*_*/         # Классы с реализацией
└── src/test/java/           # Тесты
│   └── hw_12_unit_test/
│       └── task_*_*/         # Тестовые классы
└── pom.xml                   # Maven конфигурация
└── README.md                 # Данный файл
```

## ✅ Требования к тестам

Для каждой задачи необходимо покрыть тестами:

### 🔹 Позитивные сценарии
- Корректные входные данные
- Ожидаемый результат

### 🔹 Граничные значения
- Пустые строки / null
- Нулевые значения
- Минимальные/максимальные значения

### 🔹 Негативные сценарии
- Некорректные входные данные
- Ожидаемые исключения

## 🛠️ Технологии

- **Java 17+**
- **JUnit 5** (Jupiter)
- **Maven** (для сборки и управления зависимостями)
- **Jackson** (для задачи 19)

---

## 📦 Зависимости (pom.xml)

```xml
<dependencies>
    <!-- JUnit 5 -->
    <dependency>
        <groupId>org.junit.jupiter</groupId>
        <artifactId>junit-jupiter-api</artifactId>
        <version>5.10.0</version>
        <scope>test</scope>
    </dependency>
    
    <!-- Jackson для работы с JSON -->
    <dependency>
        <groupId>com.fasterxml.jackson.core</groupId>
        <artifactId>jackson-databind</artifactId>
        <version>2.15.2</version>
    </dependency>
</dependencies>
```

## 🚀 Запуск тестов

```bash
# Запуск всех тестов
mvn test

# Запуск тестов для конкретной задачи
mvn test -Dtest=EvenCheckerTest

# Запуск через IDE (IntelliJ IDEA/Eclipse)
# ПКМ на папке test → Run Tests
```
