# Задача 10: Корзина

## Описание

В системе есть корзина.

У корзины есть:

- список товаров

У каждого товара в корзине есть:

- название
- цена
- количество

Нужно реализовать систему, которая может:

- добавить товар в корзину
- удалить товар из корзины
- изменить количество товара
- посчитать общую стоимость корзины
- показать содержимое корзины

Правила:

- если количество стало 0, товар удаляется из корзины
- общая стоимость корзины считается как сумма цена × количество для всех товаров

Пример:

В корзине:

- Телефон — 500 × 1
- Чехол — 20 × 2
- Зарядка — 30 × 1

Общая стоимость:
570
