# Задача 9: Библиотека

## Описание

В системе есть библиотека.

У библиотеки есть:

- название
- список книг

У каждой книги есть:

- название
- автор
- количество доступных экземпляров

Нужно реализовать систему, которая может:

- добавить книгу в библиотеку
- выдать книгу читателю
- вернуть книгу обратно
- показать список доступных книг

Правила:

- книгу можно выдать, только если есть хотя бы один доступный экземпляр
- при выдаче количество уменьшается на 1
- при возврате количество увеличивается на 1

Пример:

В библиотеке есть книги:

- "1984" — Оруэлл — 2
- "Мастер и Маргарита" — Булгаков — 1

После одной выдачи:

- "1984" — 1
- "Мастер и Маргарита" — 1

После возврата:

- "1984" — 2
- "Мастер и Маргарита" — 1
