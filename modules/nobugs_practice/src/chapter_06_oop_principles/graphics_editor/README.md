# Задача: Простой графический редактор (Интерфейс + Поведение)

## Описание
Создание упрощённой модели графического редактора, который может работать с разными типами фигур и объектов. Задача демонстрирует ключевую идею **полиморфизма через интерфейс**: холст (канва) хранит коллекцию объектов, реализующих общий интерфейс `Drawable`, и может отрисовать их все, не зная конкретных типов (линия, прямоугольник, текст).

## Условие
1.  Создайте **интерфейс** `Drawable` с методом `void draw()`.
2.  Реализуйте три класса, которые по-разному реализуют метод `draw()`:
    *   **`Line`** — рисует линию. Принимает координаты начала и конца (например, `x1, y1, x2, y2`). Вывод в консоль: `"Линия от (x1,y1) до (x2,y2)"`.
    *   **`Rectangle`** — рисует прямоугольник. Принимает координаты левого верхнего угла, ширину и высоту. Вывод: `"Прямоугольник в (x,y) размером width x height"`.
    *   **`Text`** — рисует текст. Принимает координаты и строку текста. Вывод: `"Текст '...' в позиции (x,y)"`.
3.  Создайте класс **`Canvas`** (холст):
    *   Поле `private List<Drawable> elements` (список объектов для отрисовки).
    *   Метод `void addElement(Drawable element)` — добавляет объект на холст.
    *   Метод `void removeElement(Drawable element)` — удаляет объект (по желанию).
    *   Метод `void renderAll()` — проходит по всем элементам в списке и вызывает у каждого метод `draw()`.
4.  В методе `main` продемонстрируйте:
    *   Создание холста.
    *   Добавление на холст разных объектов (линий, прямоугольников, текста).
    *   Вызов `renderAll()` для отрисовки всего содержимого.

## Пояснение
Интерфейс `Drawable` служит контрактом, гарантирующим наличие метода `draw()`. Класс `Canvas` работает с этим интерфейсом и абсолютно не зависит от того, какие именно объекты на нём находятся — линия, прямоугольник или текст. Это позволяет легко добавлять новые типы рисуемых объектов (например, круг или изображение) без изменения кода холста.

## Пример кода (структура)

```java
import java.util.ArrayList;
import java.util.List;

// Интерфейс для всех рисуемых объектов
interface Drawable {
    void draw();
}

// Класс Линия
class Line implements Drawable {
    private int x1, y1, x2, y2;
    
    public Line(int x1, int y1, int x2, int y2) {
        this.x1 = x1;
        this.y1 = y1;
        this.x2 = x2;
        this.y2 = y2;
    }
    
    @Override
    public void draw() {
        System.out.println("Линия от (" + x1 + "," + y1 + ") до (" + x2 + "," + y2 + ")");
    }
}

// Класс Прямоугольник
class Rectangle implements Drawable {
    private int x, y, width, height;
    
    public Rectangle(int x, int y, int width, int height) {
        this.x = x;
        this.y = y;
        this.width = width;
        this.height = height;
    }
    
    @Override
    public void draw() {
        System.out.println("Прямоугольник в (" + x + "," + y + ") размером " + width + " x " + height);
    }
}

// Класс Текст
class Text implements Drawable {
    private int x, y;
    private String content;
    
    public Text(int x, int y, String content) {
        this.x = x;
        this.y = y;
        this.content = content;
    }
    
    @Override
    public void draw() {
        System.out.println("Текст '" + content + "' в позиции (" + x + "," + y + ")");
    }
}

// Класс Холст, управляющий коллекцией рисуемых объектов
class Canvas {
    private List<Drawable> elements = new ArrayList<>();
    
    public void addElement(Drawable element) {
        elements.add(element);
        System.out.println("Элемент добавлен на холст");
    }
    
    public void removeElement(Drawable element) {
        elements.remove(element);
        System.out.println("Элемент удалён с холста");
    }
    
    public void renderAll() {
        System.out.println("\n--- Отрисовка холста ---");
        if (elements.isEmpty()) {
            System.out.println("Холст пуст");
        } else {
            for (Drawable element : elements) {
                element.draw(); // Полиморфный вызов!
            }
        }
        System.out.println("------------------------\n");
    }
}

// Демонстрация
public class Main {
    public static void main(String[] args) {
        // Создаём холст
        Canvas canvas = new Canvas();
        canvas.renderAll(); // Пустой холст
        
        // Создаём разные объекты
        Line line = new Line(10, 10, 100, 50);
        Rectangle rect = new Rectangle(20, 30, 80, 60);
        Text text = new Text(50, 70, "Привет, мир!");
        
        // Добавляем на холст
        canvas.addElement(line);
        canvas.addElement(rect);
        canvas.renderAll(); // Отрисовываем с линией и прямоугольником
        
        // Добавляем текст
        canvas.addElement(text);
        canvas.renderAll(); // Отрисовываем всё
        
        // Удаляем линию
        canvas.removeElement(line);
        canvas.renderAll(); // Отрисовываем без линии
    }
}
```

## Ключевые моменты реализации

*   **Интерфейс как контракт**: `Drawable` гарантирует, что любой объект, добавленный на холст, умеет себя рисовать через метод `draw()`.
*   **Полиморфная коллекция**: `List<Drawable>` может хранить объекты любых классов, реализующих этот интерфейс.
*   **Единообразная обработка**: Цикл `for (Drawable element : elements)` не делает никаких проверок на тип (`instanceof`). Он просто вызывает `element.draw()`, и каждый объект сам определяет, как именно себя нарисовать.
*   **Расширяемость**: Чтобы добавить новую фигуру (например, `Circle` или `Triangle`), достаточно создать новый класс, реализующий `Drawable`. Код холста остаётся неизменным.
