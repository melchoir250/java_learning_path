# Задача 28: Онлайн-магазин с корзиной (Композиция + Инкапсуляция)

## Описание проекта

Разработайте систему корзины интернет-магазина, демонстрирующую принципы композиции и инкапсуляции. Корзина содержит
список товаров, но прямой доступ к этому списку закрыт. Все операции с корзиной выполняются через специальные методы,
что позволяет контролировать изменения и обеспечивать целостность данных.

## Структура проекта

```
shopping_cart/
├── Product.java              # Класс товара
├── ShoppingCart.java         # Класс корзины
├── Main.java                 # Демонстрация работы
└── README.md                 # Описание задачи
```

## Функциональные требования

### 1. Класс `Product`

Представляет товар в магазине:

- **Поля** (все приватные):
    - `String id` - уникальный идентификатор товара
    - `String name` - название товара
    - `double price` - цена товара
    - `String category` - категория товара (опционально)
    - `int quantity` - количество на складе (опционально)
- **Конструктор**: для инициализации всех обязательных полей
- **Геттеры**: для доступа к полям товара
- **Метод `toString()`**: для удобного вывода информации о товаре
- **Метод `equals()` и `hashCode()`**: для корректного сравнения товаров (опционально)

### 2. Класс `ShoppingCart`

Представляет корзину покупок (композиция):

- **Поле**:
    - `List<Product> items` - список товаров в корзине (приватное)
- **Конструктор**: инициализирует пустой список товаров
- **Методы**:
    - `void addProduct(Product product)` - добавить товар в корзину
    - `void addProduct(Product product, int quantity)` - добавить товар с указанием количества (опционально)
    - `boolean removeProduct(Product product)` - удалить товар из корзины
    - `boolean removeProduct(String productId)` - удалить товар по ID
    - `void clear()` - очистить корзину
    - `double getTotalPrice()` - рассчитать общую стоимость корзины
    - `int getItemCount()` - получить общее количество товаров
    - `int getUniqueItemCount()` - количество уникальных товаров
    - `boolean isEmpty()` - проверить, пуста ли корзина
    - `List<Product> getItems()` - получить **копию** списка товаров (защита от изменений)
    - `boolean contains(Product product)` - проверить наличие товара

### 3. Класс `Main`

Демонстрирует работу системы:

- Создаёт несколько товаров
- Создаёт корзину
- Добавляет и удаляет товары
- Показывает общую стоимость
- Демонстрирует, что прямой доступ к списку невозможен

## Технические требования

### Инкапсуляция

- Список товаров должен быть приватным
- Нет прямого доступа к списку извне
- Метод `getItems()` возвращает копию списка, а не оригинал

### Композиция

- `ShoppingCart` содержит список товаров (has-a отношение)
- Корзина управляет жизненным циклом списка

### Контроль операций

- Добавление: проверка на null, возможно проверка дубликатов
- Удаление: проверка существования товара
- Обработка пограничных случаев

### Расчёт стоимости

- Суммирование цен всех товаров
- Учёт количества (если реализовано)

## Ключевые концепции

- **Инкапсуляция**: скрытие внутреннего состояния корзины
- **Композиция**: корзина "имеет" список товаров
- **Контроль доступа**: операции только через специальные методы
- **Защита данных**: возврат копий коллекций
