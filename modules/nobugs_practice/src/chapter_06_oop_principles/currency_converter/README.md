# Задача 39: Валюта и пересчёт (Стратегия + Интерфейс)

## Описание проекта

Разработайте систему конвертации валют, демонстрирующую паттерн "Стратегия" через интерфейс. Различные способы получения
курсов валют (фиксированные курсы, онлайн-курсы) реализуют единый интерфейс `CurrencyConverter`. Клиентский код работает
через интерфейс, что позволяет легко менять стратегию конвертации во время выполнения без изменения существующего кода.

## Структура проекта

```
currency_converter/
├── CurrencyConverter.java        # Интерфейс конвертера валют
├── FixedRateConverter.java       # Конвертер с фиксированными курсами
├── OnlineRateConverter.java      # Конвертер с онлайн-курсами
├── CurrencyService.java          # Сервис для работы с конвертацией
├── Main.java                     # Демонстрация работы
└── README.md                     # Описание задачи
```

## Функциональные требования

### 1. Интерфейс `CurrencyConverter`

Определяет контракт для всех конвертеров валют:

- **Метод `double convert(double amount, String fromCurrency, String toCurrency)`** - конвертирует сумму из одной валюты
  в другую
- **Метод `double getExchangeRate(String fromCurrency, String toCurrency)`** - возвращает курс обмена
- **Метод `Set<String> getSupportedCurrencies()`** - возвращает поддерживаемые валюты (опционально)
- **Метод `void updateRates()`** - обновляет курсы (для онлайн-конвертера)

### 2. Реализации конвертеров

#### `FixedRateConverter` (Фиксированные курсы)

- **Поля**:
    - `Map<String, Map<String, Double>> rates` - фиксированные курсы валют
    - `String baseCurrency` - базовая валюта (например, "USD")
- **Особенности**:
    - Курсы задаются в конструкторе или через конфигурацию
    - Не требуют внешних API вызовов
    - Быстрые и предсказуемые
- **Логика конвертации**:
    - Конвертирует сумму через базовую валюту
    - Поддерживает прямые кросс-курсы

#### `OnlineRateConverter` (Онлайн-курсы)

- **Поля**:
    - `String apiUrl` - URL API для получения курсов
    - `Map<String, Map<String, Double>> cache` - кэш курсов
    - `long cacheExpiryTime` - время жизни кэша
    - `long lastUpdateTime` - время последнего обновления
- **Особенности**:
    - Получает актуальные курсы из внешнего API
    - Кэширует курсы для повышения производительности
    - Может обрабатывать ошибки сети
- **Логика конвертации**:
    - Проверяет кэш, при необходимости обновляет
    - Выполняет конвертацию

### 3. Класс `CurrencyService`

Представляет сервис для работы с конвертацией валют:

- **Поле**:
    - `CurrencyConverter converter` - текущая стратегия конвертации
- **Конструктор**: принимает начальную стратегию
- **Методы**:
    - `void setConverter(CurrencyConverter converter)` - смена стратегии
    - `double convert(double amount, String from, String to)` - конвертация с текущей стратегией
    - `double getRate(String from, String to)` - получение курса
    - `void displayConversion(double amount, String from, String to)` - вывод конвертации

### 4. Класс `Main`

Демонстрирует работу системы:

- Создаёт разные стратегии конвертации
- Создаёт сервис с начальной стратегией
- Выполняет конвертации
- Меняет стратегию во время выполнения
- Показывает разницу между фиксированными и онлайн-курсами

## Технические требования

### Паттерн "Стратегия"

- Все конвертеры реализуют единый интерфейс
- `CurrencyService` работает с интерфейсом, а не с конкретными классами
- Стратегия может быть заменена во время выполнения

### Инкапсуляция

- Каждая стратегия инкапсулирует свою логику получения курсов
- Детали реализации скрыты от клиента

### Обработка ошибок

- Проверка поддержки валюты
- Выброс исключения `CurrencyNotSupportedException` при неподдерживаемой валюте
- Обработка ошибок сети для онлайн-конвертера

### Кэширование (для онлайн-конвертера)

- Хранение полученных курсов
- Автоматическое обновление при устаревании

## Ключевые концепции

- **Strategy Pattern**: выбор алгоритма конвертации во время выполнения
- **Полиморфизм**: единый интерфейс для разных стратегий
- **Слабая связанность**: сервис зависит только от интерфейса
- **Кэширование**: оптимизация производительности
