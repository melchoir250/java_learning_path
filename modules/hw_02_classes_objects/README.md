# Классы объектов в Java

Проект демонстрирует основы объектно-ориентированного программирования в Java через реализацию различных классов с использованием принципов инкапсуляции.

## Структура классов

### Car
- **Поля**: brand (String), year (int)
- **Функциональность**: конструктор, геттеры/сеттеры, вывод информации
- **Назначение**: моделирование автомобиля

### Rectangle
- **Поля**: width (double), height (double)
- **Функциональность**: геттеры, сеттер для ширины, расчет площади
- **Назначение**: геометрические вычисления

### Book
- **Поля**: title (String), author (String)
- **Функциональность**: полный набор геттеров/сеттеров, вывод информации
- **Назначение**: управление данными о книгах

### BankAccount
- **Поля**: owner (String), balance (double)
- **Функциональность**: операции пополнения и снятия, вывод баланса
- **Назначение**: моделирование банковского счета

### Point
- **Поля**: x (double), y (double)
- **Функциональность**: геттеры, сеттер для X, вывод координат
- **Назначение**: работа с координатами в пространстве

### StudentGroup
- **Поля**: groupName (String), studentCount (int)
- **Функциональность**: полный набор геттеров/сеттеров, вывод информации
- **Назначение**: управление данными учебных групп

### Circle
- **Поля**: radius (double)
- **Функциональность**: геттер/сеттер, расчет площади и длины окружности
- **Назначение**: геометрические вычисления для круга

### Teacher
- **Поля**: name (String), subject (String)
- **Функциональность**: полный набор геттеров/сеттеров, вывод информации
- **Назначение**: управление данными преподавателей

### Product
- **Поля**: name (String), price (double)
- **Функциональность**: сеттер для цены, применение скидок, вывод информации
- **Назначение**: управление товарами и ценами

### Laptop
- **Поля**: brand (String), price (double)
- **Функциональность**: полный набор геттеров/сеттеров, вывод информации
- **Назначение**: управление данными о ноутбуках

## Принципы реализации

- **Инкапсуляция**: все поля приватные с контролируемым доступом через методы
- **Геттеры/сеттеры**: управление доступом к данным класса
- **Конструкторы**: инициализация объектов при создании
- **Специализированные методы**: бизнес-логика соответствующих предметных областей

Каждый класс представляет отдельную сущность с четко определенным набором полей и методов, демонстрируя различные аспекты проектирования объектов в Java.
