# Домашнее задание: Коллекции в Java

## Описание проекта

Этот проект посвящён практическому освоению Java Collections Framework. В рамках заданий вы будете работать с основными интерфейсами и реализациями коллекций: List, Set, Map, Queue и Deque. Каждая задача направлена на понимание особенностей работы конкретной структуры данных, её преимуществ и типичных сценариев использования.

## Структура заданий

### ArrayList (минимум 2 задачи)
1. Создайте ArrayList из 5 чисел. Добавьте ещё одно число в конец. Выведите весь список.
2. Напишите программу, которая выводит все чётные числа из ArrayList.
3. Создайте ArrayList из строк. Найдите в нём самую длинную строку и выведите её.
4. Создайте ArrayList из целых чисел. Вычислите и выведите сумму всех чисел.
5. Создайте ArrayList из целых чисел. Найдите и выведите максимальное число.

### LinkedList (минимум 2 задачи)
1. Создайте LinkedList из 5 строк. Выведите все элементы.
2. Реализуйте очередь задач с LinkedList. Добавьте 3 задачи и обработайте их по порядку.
3. Создайте LinkedList из строк. Выведите первый и последний элементы.
4. Создайте LinkedList из целых чисел. Вычислите сумму элементов.
5. Используйте ListIterator для прохода по LinkedList в обоих направлениях.

### HashSet (минимум 2 задачи)
1. Создайте HashSet из 5 чисел и выведите его содержимое.
2. Добавьте в HashSet 10 чисел. Проверьте наличие заданного числа.
3. Реализуйте метод, который принимает List<String> и возвращает Set<String> без дубликатов.
4. Создайте HashSet с именами. Проверьте, содержится ли ваше имя в множестве.

### LinkedHashSet (минимум 1 задача)
1. Создайте LinkedHashSet из 5 строк. Проверьте порядок элементов при выводе.
2. Напишите метод, который добавляет элемент в LinkedHashSet без дубликатов.

### TreeSet (минимум 1 задача)
1. Создайте TreeSet из 5 чисел и выведите (обратите внимание на порядок).
2. Добавьте числа в TreeSet без дубликатов.
3. Найдите ближайшее большее и меньшее число к заданному в TreeSet.

### HashMap (минимум 2 задачи)
1. Создайте HashMap<String, Integer>, добавьте 5 пар (имя–возраст) и выведите.
2. Проверьте наличие определённого имени в HashMap.
3. Выведите всех пользователей младше 18 лет из HashMap.

### LinkedHashMap (минимум 1 задача)
1. Создайте LinkedHashMap и добавьте 5 элементов. Выведите в порядке добавления.
2. Реализуйте телефонную книгу с LinkedHashMap.
3. Создайте LinkedHashMap для истории просмотров (максимум 10 элементов).

### TreeMap (минимум 1 задача)
1. Создайте TreeMap, добавьте 5 ключей (имена) и значений (баллы). Выведите отсортированные данные.
2. Найдите минимальный и максимальный ключ в TreeMap.
3. Реализуйте TreeMap для сотрудников с поиском ближайшего большего ID.

### PriorityQueue (1 задача)
1. Создайте PriorityQueue, добавьте 5 чисел. Выведите в порядке удаления.

### ArrayDeque (минимум 1 задача)
1. Создайте ArrayDeque, добавьте 5 элементов и выведите.
2. Используйте ArrayDeque как стек.
3. Используйте ArrayDeque как двустороннюю очередь.

## Ключевые концепции

### Основные интерфейсы коллекций
- **List** — упорядоченные коллекции с доступом по индексу
- **Set** — коллекции уникальных элементов
- **Map** — пары ключ-значение
- **Queue** — очереди (FIFO)
- **Deque** — двусторонние очереди

### Реализации и их особенности
- **ArrayList vs LinkedList** — массив vs связанный список
- **HashSet vs LinkedHashSet vs TreeSet** — хэш-таблица vs упорядоченный vs отсортированный
- **HashMap vs LinkedHashMap vs TreeMap** — аналогично для Map
- **PriorityQueue** — очередь с приоритетами
- **ArrayDeque** — эффективная двусторонняя очередь

### Практические навыки
- Выбор подходящей коллекции для задачи
- Работа с итераторами
- Преобразования между типами коллекций
- Реализация алгоритмов с использованием коллекций
- Отладка и оптимизация работы с данными
