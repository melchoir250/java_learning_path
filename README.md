# Java Learning Path

Репозиторий с учебными проектами и домашними заданиями по Java. Каждая директория содержит решение отдельной задачи с фокусом на конкретные темы программирования.

## Структура проектов

```
java_learning_path/modules/
├── math_operations/           # Математические операции и утилиты
├── object_classes/           # Классы и объекты
├── static_final_modifiers/   # Модификаторы static и final
├── exceptions_generics/    # Исключения и дженерики ⭐ НОВОЕ
├── conditionals_loops/       # Условные операторы и циклы
├── oop_principles/           # Принципы ООП
├── collections/              # Коллекции
├── nobugs_practice/          # Дополнительные задачи из книги NoBugs
└── [следующие проекты...]
```

Каждый проект представляет собой автономный модуль с полной реализацией:

- Исходный код в директории `src/`
- Документация в `README.md`

## Темы и содержание

### Math Operations
- Базовые арифметические операции
- Геометрические вычисления
- Конвертация единиц измерения
- Температурные преобразования

### Object Classes
- Классы Car, Rectangle, Book, BankAccount
- Принципы инкапсуляции
- Геттеры и сеттеры
- Специализированные методы

### Static & Final Modifiers
- Статические поля и методы
- Финальные переменные и константы
- Модификаторы доступа
- Практическое применение модификаторов

### Conditionals & Loops
- Условные операторы if-else и вложенные проверки
- Конструкция switch с примерами на обычных значениях и enum
- Циклы:
- for — повторение действий с известным числом итераций
- while — выполнение пока условие истинно
- do-while — гарантированное выполнение тела цикла минимум один раз
- Управление циклами: break и continue

+ ### Exceptions & Generics ⭐ НОВОЕ
+ Обработка проверяемых и непроверяемых исключений
+ Создание пользовательских исключений
+ Дженерик-классы и методы
+ Параметризация с несколькими типами
+ Ограничения типов (extends, super)
+ Дженерики в коллекциях


### OOP Principles Practice

- Инкапсуляция: скрытие данных через приватные поля и управление доступом через методы
- Наследование: создание базовых классов и расширение поведения в потомках
- Полиморфизм: переопределение методов и единый интерфейс для разных объектов
- Абстракция: выделение общего поведения через абстрактные классы и интерфейсы
- Проектирование систем с единым интерфейсом для разных типов объектов

## Использование

Каждый проект независим и может быть запущен отдельно:

```bash
cd [название_проекта]/src
javac *.java
java Main.java
```
