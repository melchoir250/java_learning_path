# NoBugs Practice

Практические задачи из книги "NoBugs: Экспресс Java решебник" для системного закрепления тем языка Java.

## Структура практики

```
nobugs_practice/
├── chapter_01_first_program/          # Первая Java программа
├── chapter_02_classes_objects/        # Первое создание классов и объектов
├── chapter_03_static_final/           # Static, final, модификаторы доступа
├── chapter_04_conditions_loops/       # Условные операторы и циклы
├── chapter_05_oop_design/             # Алгоритм проектирования задач в ООП
├── chapter_06_oop_principles/         # ООП принципы в Java
├── chapter_07_data_structures/        # Алгоритм выбора структуры данных
├── chapter_08_collections/            # Коллекции в Java
├── chapter_09_object_methods/         # Класс объекта и его методы
├── chapter_10_exceptions_generics/    # Исключения и дженерики
├── chapter_11_functional/             # Функциональные интерфейсы и Stream API
├── chapter_12_multithreading/         # Многопоточность
├── chapter_13_testing/                # Пишем тесты на код
├── chapter_14_problem_solving/        # Решение проблем
└── chapter_15_complex_tasks/          # Комплексные задачи на все темы
```

## Содержание разделов

### Основы Java
- **Глава 1**: Первая Java программа - базовый синтаксис и структура
- **Глава 2**: Классы и объекты - основы ООП
- **Глава 3**: Модификаторы доступа - static, final, области видимости

### Базовые конструкции
- **Глава 4**: Условные операторы и циклы - управляющие конструкции
- **Глава 5**: Проектирование в ООП - архитектура приложений
- **Глава 6**: Принципы ООП - наследование, полиморфизм, инкапсуляция

### Продвинутые темы
- **Глава 7**: Структуры данных - выбор оптимальных решений
- **Глава 8**: Коллекции - List, Set, Map, итераторы
- **Глава 9**: Методы объектов - проектирование поведения классов

### Современный Java
- **Глава 10**: Исключения и дженерики - обработка ошибок, параметризованные типы
- **Глава 11**: Функциональное программирование - лямбды, Stream API
- **Глава 12**: Многопоточность - параллельное выполнение

### Профессиональные навыки
- **Глава 13**: Тестирование - модульные тесты, JUnit
- **Глава 14**: Решение проблем - алгоритмическое мышление
- **Глава 15**: Комплексные задачи - интеграция всех изученных тем

## Формат работы

Каждая глава содержит:
- Практические задачи с постепенным усложнением
- Примеры реализации и тестирования
- Рекомендации по проектированию решений
- Критерии оценки качества кода

## Цели практики

- Системное освоение Java от базового синтаксиса до продвинутых тем
- Развитие навыков объектно-ориентированного проектирования
- Приобретение опыта решения реальных программистских задач
